Across-Trials Comparisons of Clinical Data

To perform an across-trials comparison of clinical data:

1.  Run tranSMART, then click the Dataset Explorer tab.

2.  In the left pane, click the new Across Trials tab:

3.  Open the Across Trials node, then open the Clinical Data node. 

A list of Clinical Data nodes appears.  The nodes represent all the types of clinical data that can be used in across-trials comparisons:

4.  Open the node representing the clinical data to use in the comparison.

In the following example, the Psoriasis Area Severity Index (PASI) node is opened:

5.  Open the timepoint to use as the starting timepoint for the comparison – for example, Baseline.

6.  Drag the studies to use in the comparison into the same subset box in Subset 1.

In the following example, two studies have been dragged into Subset 1 from the Baseline timepoint:

7.  Open the timepoint to use as the ending timepoint for the comparison.

8.  Drag the studies to use in the comparison into the same subset box in Subset 2.

In the following example, the same two studies have been dragged into Subset 2 from the Week 24 timepoint:

9.  Click Generate Summary Statistics.

The comparison appears at the bottom of the Results/Analysis view.
