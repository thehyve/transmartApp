GeneGo Tab

When you click the GeneGo tab after a tranSMART search, the search string is passed to GeneGo, and the GeneGo search against the tranSMART      search string begins immediately.

Note:  If a security warning appears when you click this tab, see Security Warning When Selecting Pictor, ResNet, or GeneGO.

tranSMART displays the GeneGo user interface in the tranSMART results pane.  You can view the initial search results there, and use the GeneGo controls to perform additional GeneGo searches.

Note:  If you are prompted for login credentials after clicking the GeneGo tab, hover the mouse cursor over the GeneGo tab to see the credentials you need, then supply them in the Authentication Required dialog.
