Search Tasks

tranSMART provides a Google-like interface for searching across internal Johnson & Johnson data sources as well as external data sources with a single query, based on one or more search filters that you define.

A search filter is one of the following:

■    The name of a biomedical concept such as a gene, pathway, compound, disease, or other item of medical interest. These filter names are pre-defined in tranSMART.  You can browse lists of these filter names and select the filter you want, or type part of a filter name in the Search field, causing tranSMART to display a list of filters that begin with the text you type.

■    Any text that you type into the Search field and then click the Search button or press the Enter key.  By clicking Search or pressing Enter rather than selecting from a list of pre-defined filter names, you instruct tranSMART to limit its search to internal Johnson & Johnson document repositories, looking for the exact text you typed (not case-sensitive).

You can base your search on a single search filter or on a multi-filter search string.
