Asynchronous Operations

You may run multiple advanced workflow operations in Dataset Explorer asynchronously. Analyses run in the background of the program, allowing you to use other features of the tranSMART application or to perform additional analyses simultaneously within Dataset Explorer.

To run advanced workflow(s) in the background of Dataset Explorer:

1.  Select the type of advanced workflow to run from the Advanced menu.

2.  When the Job Status dialog box appears, select Run in Background:

After selecting Run in Background, you will see the status of your job in the lower right corner of your browser:

If you run multiple jobs in the background, the status will cycle through each job in the order that the jobs were started.

3.  Click OK to view results:

Note:  For smaller analyses you will not see the Complete dialog box – the visualizer will appear automatically.
