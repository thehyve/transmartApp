Literature Tab

This result category contains hits from a set of curated articles within a set of selected diseases. Currently, the set includes a selection of genes of particular interest to J&J in different cancers a set of selected genes and their roles in a set of respiratory diseases (asthma, IPF, COPD, etc. The  curation services are provided by Jubilant Biosys.

Sources of the articles include PubMed and ClinicalTrials.gov.

Click the Literature tab to display the results in this category. The following figure shows the controls that appear at the top of the results list:

These controls give you access to the following views and operations:

■    Show Filters button – Define additional filters to further refine the search results.

■    Export Results button – Write curation data to Microsoft Excel.

■    Export ResNet button– Write selected facts from Jubilant curated content to Ariadne Genomics’ Pathway Studio for network visualization and further in-depth analysis.

■    Results for dropdown – Specify the type of literature results you want to see in the categories of oncology and asthma.  In the figure above, matches for the genes in the gene signature were found in the categories of oncology inhibitors and asthma inhibitors.  The Results for dropdown lets you select and view results for one of those categories.
