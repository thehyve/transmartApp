mRNA Profiles Tab

This result category allows mining of the expression levels of a gene or a set of genes in the context of a set of curated public studies involving different cancers, as collated and curated by Dana Farber Cancer Institute.

To see the results in this category, click the mRNA Profiles tab.

The number in parentheses on the tab represents the total number of studies that matched the search criteria.

The following figure shows the layout of the mRNA profile result window. The list of genes in the Gene dropdown are associated with the gene signature TLR3 in HT29 cells (the subject of the search). As you select different genes in the Gene dropdown, the diseases and probe sets that have meaning for the selected gene become available in the Disease and Probe Set dropdowns.
