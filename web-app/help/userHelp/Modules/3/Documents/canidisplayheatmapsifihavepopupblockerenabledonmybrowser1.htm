Can I display heat maps if I have pop-up blocker enabled on my browser?

Yes. To allow heat maps to be displayed in Microsoft Internet Explorer 8 if you have pop-up blocker enabled, do the following:

 

1.  Internet Explorer, click the Tools menu, then click Internet Options.

2.  Click the Privacy tab.

3.  In the Pop-up Blocker group box, click Settings.

4.  In Address of website to allow, type the following:

jnj.com

 

5.  Click Add.

6.  Click Close, then click OK to close the Internet Options dialog.

Note:  If your security software blocks pop-ups, refer to the documentation to learn how to allow pop-ups from the tranSMART site.
