Can I remove some of a pathway’s genes from the search filter?

Yes. To split a pathway into its component genes, click the green Split icon ( ) after the pathway name on the Edit Filters dialog.  You can then delete one or more of the pathway’s genes from the search filter.  For example:

 

1.  Click the filter category pathway in the tranSMART search window:

2.  Type il4 in the Search field, then select Pathway>GeneGO> Il4 Signaling Pathway.

3.  Click the advanced button to display the Edit Filters dialog.

4.  Click the Split icon ( ) after the pathway name:

The Edit Filters dialog now looks as follows:

5.  To remove one or more genes from the search filter, click the red x () after the name of each gene to remove.
