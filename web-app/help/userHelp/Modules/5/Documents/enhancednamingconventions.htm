Enhanced Naming Conventions

The following name structures have changed:

Patient IDs in Heat Maps

Patient IDs now have three parts: The patient’s Dataset Explorer subset (S1 or S2), the study ID, and the ID of the patient as assigned for the study, as shown below.  If the patients’ study ID is not available, the patients’ tranSMART record ID is used instead, as in past releases.

Johnson & Johnson Clinical Trial Names

Clinical trial names that appear in Dataset Explorer (and in the table Clinical Trials) now have the following structure:

TrialID_DrugName_Condition

For example:

C0168T30_Infliximab_Heart Failure

In some cases, TrialID might include the drug ID.
