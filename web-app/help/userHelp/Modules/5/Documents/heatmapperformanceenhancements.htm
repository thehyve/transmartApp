Heat Map Performance Enhancements

Performance enhancements that speed up the time required to display Dataset Explorer heat maps have been introduced in this release.  Enhancements include individual user logins to the Broad Institute’s GenePattern server (which generates the heat maps) rather than a single shared login.  These and other performance enhancements operate “under the covers” and require no user action.

Heat maps involving very large datasets can nevertheless require several minutes to be displayed.  In past releases, there was no visual feedback between the time you clicked Run Workflow to generate a heat map and the time the GenePattern Launching Visualizer dialog appeared.  You will now see a Workflow Status dialog during that interval.  This dialog displays the tasks that are being performed to display the heat map.
