Online Help

tranSMART documentation now includes a complete online Help system.  Online Help gives you the following advantages over documentation in PDF format:

■    Context-sensitivity

Clicking this question-mark button (  ) on a tranSMART window or dialog box will immediately display the associated Help topic. 

Currently, context-sensitive Help is available only for the tranSMART search window and associated dialog boxes, and the Dataset Explorer window and associated dialog boxes.  However, the Help system contains the entire tranSMART document set.
