View a Heatmap of Analysis Data

A heatmap represents gene expression intensity values for each subject in a cohort for a given probe set and gene. Each value is represented on a scale from most down-regulated (-1) to most up-regulated (1).

Heatmap rows are sorted by probe fold change value, from highest to lowest.

To view a heatmap for an analysis:

 

1.  In Search Results, click the name of the study that contains the analysis.

2.  Click the analysis name.

The heatmap is the default visualization:

 

Click a task below for information on performing that task in Heatmap view:

Set Heatmap Options

Click the Heatmap Options button to change the way the heatmap is displayed:

■    Cell size. Move the slider bubble to the right to make the blocks larger or to the left to make the blocks smaller.

■    Color range. Move the slider bubbles left or right to change the color intensity of the heatmap blocks.

■    Probes/Page. Select the number of heatmap rows to display at a time.

This setting overrides the global Probes/Page option for this analysis’ heatmap only.

Export Analysis Data to a CSV File

To view the data that the heatmap represents, click the Export button, then select the option you want:

To open the CSV file, click the button that appears at the lower left corner of the browser, or click the down-arrow to the right of the button for options.

Exported data includes gene, fold change, TEA value, preferred p-value, intensity value, and test subject ID for the specified probe ID.

Export the Heatmap to a Graphics File

To export the heatmap to a graphics file, click Export, then click Export image:

To open the graphics file, click the button that appears at the lower left corner of the browser, or click the down-arrow to the right of the button for options.

The heatmap on the current page will be exported to the file. To change the number of heatmap rows you can export to the image file, see Set Heatmap Options above.

View the Scale of Gene Expression Intensity

To view the scale of gene expression intensity for a particular probe set, gene, and study subject, hover the mouse pointer over the corresponding heatmap block:

The ID of the study subject is not displayed.

Note that gene expression intensity is represented on a scale of -1 (maximum down-regulation) through 1 (maximum up-regulation). To see the actual gene expression intensity values, along with the IDs of the study subjects associated with the intensity values, export the data to a CSV file.
