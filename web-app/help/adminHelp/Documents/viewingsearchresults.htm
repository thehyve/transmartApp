Viewing Search Results

Search Results view displays a list of all studies that match the search query defined in Active Filters.

You can perform the following tasks in Search Results view:

■    Browse the list of studies, view study metadata, and open the list of analyses of the study.

See Browse the Study List.

■    Browse the list of analyses of a study, view analysis metadata, and select an analysis for inclusion in a cross-trial analysis.

See Browse the Analysis List.

■    Generate a heatmap for an analysis.

See View a Heatmap of Analysis Data.

■    Generate a Box Plot for an analysis.

See View a Box Plot of Analysis Data.

■    Generate a Line Plot for a Time Course analysis.

See View a Line Plot of Time Course Analysis Data.

■    Set Global Options

See Set Global Options.
