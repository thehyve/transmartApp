View a Line Plot of Time Course Analysis Data

Line plots are supported for analyses of type Time Course. You can select this type of analysis from the Analysis filter browser.

To view a line plot for an analysis:

 

1.  In Search Results, click the name of the study that contains the analysis.

2.  Click the analysis name.

3.  Click Lineplot.

Note that the cohort descriptions use bold type to highlight the times when the data for each cohort was collected:

View Data Intensity Values

The data points in the line plot are the mean log2 intensity values. The horizontal bars above and below a data point indicate the mean log2 intensity value plus or minus the standard error.

Hover the mouse pointer over one of the data points in the line plot to view this data:

Set Line Plot Options

Click the Lineplot Options button to change the way the line plot is displayed:

■    Probeset. Select the probe set and gene to represent in the line plot.

The list of probe set/gene combinations is determined by the applicable Probes/Page setting for this analysis, as defined in Heatmap view (see Set Heatmap Options).

■    Range. Select Manual to specify the range of gene expression intensity values for the line plot to represent, or select Auto to let tranSMART set the range.

The smaller the range, the greater the difference between the heights of the data points on the line plot.

Export Analysis Data to a CSV File

To view the data that the line plot represents, click the Export button, then click Export data:

To open the CSV file, click the button that appears at the lower left corner of the browser, or click the down-arrow to the right of the button for options.

Exported data includes gene, fold change, TEA value, preferred p-value, intensity value, and test subject ID for the probe set and gene indicated on the line plot for the specified probe ID.

Export the Line Plot to a Graphics File

To export the line plot to a graphics file, click Export, then click Export image:

To open the graphics file, click the button that appears at the lower left corner of the browser, or click the down-arrow to the right of the button for options.
