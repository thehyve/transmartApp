Faceted Search

Faceted Search provides a fine-grained search capability into studies and analyses of interest. By providing search keywords or selecting search filters from filter lists (such as Disease, Sample Type, Therapeutic Areas, Treatment), you can quickly construct a search query that focuses on one or more specific areas of interest, and then run the query against a single analysis or across multiple analyses.

Query results can be generated to visualizations such as heatmaps, box plots, and line plots, and the data can be exported to a CSV file.
