View Gene Charts and Box Plots

To view a gene chart for a particular gene or protein in any of the selected analyses:

1.  Type part or all of the gene or protein name in the Search for genes, pathways, or gene signatures field.

2.  Click the name in the dropdown list.

3.  Optionally, repeat these steps to generate a gene chart for additional genes and proteins.

Generate a Box Plot

To generate a box plot for a gene or protein:

1.  Hover the mouse pointer over the corresponding gene chart.

2.  Click view boxplot.

A box plot appears for each analysis:

3.  To see the data that a particular box plot represents, hover the mouse pointer over a box in the box plot.

For information about the data that a box plot represents, see View Intensity Values.

4.  Optionally, to save the box plots to a graphics file, click Export Box Plot as an Image.

To open the graphics file, click the button that appears at the lower left corner of the browser, or click the down-arrow to the right of the button for options.

View the Gene or Protein in a Heatmap for a Particular Analysis

To view the intensity values for a selected gene or protein in a heatmap for a particular analysis:

1.  Open the analysis of interest by clicking the analysis name in the Selected Analyses section.

2.  Click the Show Selected Genes button:

If the cross-trial analysis definition includes any pathways, gene signatures, or gene lists, the heatmap will include genes from those items as well. For more information, see View Cross-Trial Analysis Heatmaps.

Remove a Gene Chart

To remove a gene chart from the cross-trial analysis:

1.  Hover the mouse pointer over the gene chart to remove.
