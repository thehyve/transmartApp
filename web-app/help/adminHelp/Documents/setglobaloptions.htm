Set Global Options

Global options apply to all studies and analyses in Search Results view.

You set global options by clicking the Options button on the button bar:

You can set the following global options:

■    Probes/Page. With heatmaps, this setting specifies the number of heatmap rows that can be displayed at a time. This number can be overridden for an individual analysis’ heatmap via the Heatmap Options button.

■    Show only significant results. If this check box is selected, the only data that will be displayed for an analysis will be data that meets or exceeds the QA Criteria threshold in the analysis metadata.
