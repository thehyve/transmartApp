View Cross-Trial Analysis Heatmaps

A heatmap for a cross-trial analysis contains one column for each selected analysis. Each box in the heatmap represents the probe with the most significant result for a given analysis.

The most significant result for an analysis is determined by the following criteria:

■    The minimum p-value for all probe results in the analysis.

■    If p-value is not available, the maximum fold change for all probe results in the analysis.

To view a cross-trial analysis heatmap:

1.  Type part or all of a pathway, gene signature, or gene list name in the Search for genes, pathways, or gene signatures field.

2.  Click the name of the item of interest in the dropdown list.

3.  Optionally, repeat these steps to generate a heatmap for additional pathways, gene signatures, and gene lists.

The genes are sorted alphabetically.

Keep in mind that a cross-trial analysis has two kinds of heatmaps:

■    One that represents all selected analyses (like the one shown above). This heatmap is located in the top part of the cross-trial analysis display under the Heatmaps button.

■    One for each of the individual analyses. These are displayed when you click the name of an analysis in the Selected Analyses section.

 

The number of rows in a cross-trial analysis heatmap and in the heatmaps displayed for selected analyses are not affected by the global Probes/Page option and cannot be changed.

View the Genes in the Pathway, Gene Signature, or Gene List in a Heatmap for a Particular Analysis

To view the genes in the selected pathway, gene signature, or gene list in a heatmap for a particular analysis:

1.  Open the analysis of interest by clicking the analysis name in the Selected Analyses section.

2.  Click the Show Selected Genes button:

If the cross-trial analysis definition includes any selected individual genes or proteins, the heatmap will include those items as well. For more information, see View Gene Charts and Box Plots.
