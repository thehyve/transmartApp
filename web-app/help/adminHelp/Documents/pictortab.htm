Pictor Tab

This tab displays the Johnson & Johnson Pictor site in the tranSMART results pane. 

If a security warning appears when you click this tab, see Security Warning When Selecting Pictor or GeneGO.

When you click this tab after a search against a gene, pathway, gene signature or gene list, tranSMART submits the gene symbols to Pictor.  Pictor then returns any matches it finds in the pathway databases it searches.

Pictor is essentially a database of biologically meaningful gene lists. When the list does not contain expression data, the submitted genes are compared to all Pictor lists. The Pictor lists which correspond most to the submitted genes are reported in a table, ranked by hypergeometric distribution p-value statistics. When expression data is included in the analysis, Pictor lists are ranked by MeanLogP statistics.

For further information, please see the Pictor documentation.
