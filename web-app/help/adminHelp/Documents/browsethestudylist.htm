Browse the Study List

Before a search query is defined in Active Filters, all studies are listed in Search Results view. You can view the entire list of studies using the scroll bar on the page.

As you add search filters to the Active Filters area, the studies that appear in the list, and the list of analyses that can be opened for a study, are determined by the search filters you define.

You can perform the following tasks for a study:

■    View metadata for the study.

To do so, click the icon or number to the left of the study name:

Information about the study is displayed, such as the description of the study, the study owner or primary investigator, and study design.

■    Display all analyses for the study in the tranSMART Search page.

To do so, display the study metadata as shown above, then click Search analyzed Data:

This action closes the Faceted Search page. If you navigate back to the Faceted Search page using the browser’s Back function or any other method, any search query or other settings that had been defined on the Faceted Search page are lost.

■    Open the list of analyses for a study.

To do so, click the n analyses found button below the study name:

To close the list of analyses, click n analyses found again.
