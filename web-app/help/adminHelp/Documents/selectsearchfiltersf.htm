Select Search Filters from the Filter Browser

The Filter Browser lets you select one or more search filters to include in your search query.

The following figure shows the types of filters available through the Filter Browser.

Descriptions of these filters are in the table in section Defining Search Filters.

tranSMART adds all of the search filters that you specify, including those you define through the keyword search field (see Defining Search Filters), to the Active Filters area (see Manage Active Filters).

The only filters that appear in the browsers are those that match data in the data warehouse.

To select one or more filters from the Filter Browser:

1.  Under Filter Browser, click one of the filter types to display the list of filters in the browser.

In the figure below, the Disease browser is selected:

The selected browser opens, displaying the filters you can select.

A number appears in parentheses after each filter, showing the number of analyses that match all the filters in the Active Filters area. As you narrow your search results by defining more filters, the number of analyses for each filter may decrease.

2.  Select one or more of the filters in the browser. Note that some filters have sub-filters:

Grayed-out filters no longer match any data in the data warehouse (based on all the filters in Active Filters) and are not selectable from the browser.

3.  Optionally, select more filters within this browser or other browsers.

As you select each filter from a browser, tranSMART adds the filter to the Active Filters area and refreshes the Search Results.
